# WEB-AGENT

WEB-AGENT — основной модуль проекта, реализующий клиента для взаимодействия с Web API.

---

# Принцип работы агента

Агент выполняет следующий цикл:

1. Отправляет запрос на получение задачи

```
POST /app/webagent1/api/wa_task/
```

2. Сервер отвечает:

* WAIT — задач нет
* RUN — задача получена

3. Агент выполняет задание

4. Отправляет результат выполнения

```
POST /app/webagent1/api/wa_result/
```
