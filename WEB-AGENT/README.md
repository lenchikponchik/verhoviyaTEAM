# WEB-AGENT

Кроссплатформенный Web-агент для удалённого выполнения задач по протоколу HTTP/HTTPS.

Проект разрабатывается в рамках лабораторных работ и направлен на создание модульного, расширяемого и устойчивого к ошибкам сетевого агента.

## 🎯 Цель проекта

Разработка фонового приложения (агента), которое:

- регистрируется на сервере
- получает задания по HTTP/HTTPS
- выполняет команды или запускает программы
- передаёт результаты выполнения на сервер
- ведёт журнал действий
- поддерживает параллельную работу нескольких агентов

Поддерживаемые платформы:
- Windows
- Linux
- macOS

## 🛠 Требования

- C++17

---

## 🚀 Сборка проекта

```bash
mkdir build
cd build
cmake ..
make
````

После сборки будет создан исполняемый файл:

```
web_agent
```

---

## ▶ Запуск

```bash
./web_agent
```

---

## 🧪 Запуск тестов

